# Password Breach Impact Analyzer

A Java Swing desktop application that analyzes password security by combining SHA-256 hashing, password strength analysis, and graph-based relationship visualization.

## Features

- User registration with password validation
- Password strength classification (Weak / Medium / Strong)
- SHA-256 password hashing with unique salts
- Graph-based visualization of similar passwords
- Interactive security dashboard
- User security report with hash snippets

## Tech Stack

- Java
- Java Swing
- SHA-256
- Graph Data Structures
- Object-Oriented Programming
